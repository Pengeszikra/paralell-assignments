## Prerequisite:
  - npm
  - yarn
  - serve
  - modern browser

## Terminal commands (bash, gitbash):

Development:
```bash
yarn start
```

Build:
```bash
yarn build
serve ./dist
```

Test:
```bash
yarn test
```
